Налаштування для розгортання на Vercel:

1. Створено vercel.json - файл конфігурації для Vercel
2. Створено api/index.ts - точка входу для API на Vercel
3. База даних PostgreSQL готова

ВАЖЛИВО: Для роботи на Vercel потрібно:

1. Завантажити весь проект до GitHub репозиторію
2. Підключити GitHub репозиторій до Vercel
3. Налаштувати змінні середовища в Vercel:
   - DATABASE_URL (з вашої поточної бази даних)

4. Vercel автоматично виявить React проект і зібере його

ДОДАТКОВІ КРОКИ:
- Можна змінити зовнішні зображення на власні для кращої стабільності
- Переконатися що база даних доступна ззовні
